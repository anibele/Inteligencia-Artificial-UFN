Um sistema sempre é baseado em dois aspectos
aspecto estrutural (Constituição, características) x aspecto funcional (serve para qual propósito)

*Características que definem um comportamento inteligente:
    -> Guardar Base de Conhecimento/Experiência (que pode ser 'Estruturada' ou 'Não Estruturada') - Prolog, SMA, RNA;
    -> Capacidade de Raciocínio = Deliberação - Método de Busca, Prolog, Algoritmos Genéticos, SMA, RNA
        - Raciocínio Dedutivo (do geral ao específico)
        - Raciocínio Indutivo (específico leva ao geral)
    -> Reconhecimento de Padrões (Aprendizado de Máquina) - SMA, RNA;

*Usos da IA para resolver problemas:
    Pode ser por Diagnóstico -> Por reconhecer Padrões
    ou
    Pode ser por Empacotamento/Caminho ->  Por tentativa e erro

*Estado -> São Atributos e um objeto
*Regras de transição -> Métodos
*Restrições -> Condições / if
*Lista de Visitados
*Função Objetivo
